What is CSS?

Introduction to CSS

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML. It allows web developers to control the layout, appearance, and style of multiple web pages all at once.

Key Features of CSS

  • CSS enables developers to separate content from presentation, making it easier to maintain and update websites.
  • It provides a wide range of styling options, including colors, fonts, spacing, borders, and backgrounds.
  • CSS allows for responsive design, enabling websites to adapt and look good on different devices and screen sizes.
  • It supports the use of selectors to target specific HTML elements and apply styles accordingly.
  • CSS can be used alongside HTML and JavaScript to create interactive and visually appealing web experiences.

CSS Example


        /* CSS code */
        h1 {
          color: #333;
          font-size: 2rem;
        }

        p {
          color: #666;
          line-height: 1.5;
        }
